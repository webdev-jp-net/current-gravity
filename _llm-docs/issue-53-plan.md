# Issue #53 プラン: パーソナルプロットの回答を4段階から5段階に変更（3段階目を加点0）

## 与件
- 回答スケールを**4段階 → 5段階**に変更する。
- **3段階目に加点0**のポイントを設置する（中央を「どちらでもない」にする）。

---

## 現状

| 項目 | 内容 |
|------|------|
| スケール値 | `[-2, -1, 1, 2]`（4択・0なし） |
| 定義箇所 | `src/app/personal-plot/_parts/view.tsx` 31行目 |
| 送信処理 | `usePersonalPlot.ts` で `answers[q.id]` を orientation ごとに合算 → ownership / consensus / diversity / identityFusion |
| 指標範囲 | 各指標は質問5問×値 → 合計 -10 〜 +10。`useHome.ts` の `isPersonComplete` で -10〜10 を許容 |

---

## 変更方針

- スケールを **`[-2, -1, 0, 1, 2]`** に変更する。
  - 1番目: -2、2番目: -1、**3番目: 0（加点0）**、4番目: 1、5番目: 2。
- 合計値の範囲は従来どおり -10〜10のまま（0の追加で範囲は変わらない）。バリデーション・Matrix 計算の変更は不要。
