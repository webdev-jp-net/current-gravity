# **設問定義と概念詳細仕様**

このドキュメントは、CURRENT GRAVITYを構成する20の設問について、その基本データと詳細なコンセプトを定義したものです。AIエージェントがキャラクターシミュレーションや設問のブラッシュアップを行う際の絶対的な参照基準として機能します。

## 各設問のコンセプト詳細

### [valueLocus-1]

- **質問文**: 反対されることよりも矛盾があることのほうが気持ち悪い。
- **評価ラベル**: [max]正しいことが優先/[min]まるく収まることが優先
- **コンセプト**: 対人摩擦よりも、論理や構造の整合性が崩れていることを優先的に不快と感じる判断基準。

### [valueLocus-2]

- **質問文**: レポートや記事は、自分が得た意味や納得を取りこぼさないよう正確に残したい。
- **評価ラベル**: [max]正確に残す/[min]わかりやすく省略する
- **コンセプト**: 読み手のリアクションを引き出す調整よりも、自分が意味を感じた内容を残すことを優先しやすい志向。

### [valueLocus-3]

- **質問文**: 勢いが出たときや締め切り前に一気に捗ることが多い。同じペースの進捗をつくるのは窮屈。
- **評価ラベル**: [max]わかる、ムラが出る/[min]割り切って進めるのは得意
- **コンセプト**: 納得や熱量があがると急にエンジンがかかる志向。安定した進捗で外部を安心させることを忘れがちになる。

### [valueLocus-4]

- **質問文**: クイズや謎解き、まちがいさがしなどを、ヒントや攻略方法をみたり他の人が教えてくれて正解するとちょっとくやしい。
- **評価ラベル**: [max]自力で解く手応えがほしい/[min]正解できるなら気にならない
- **コンセプト**: 正解そのものよりも、過程や納得感に向き合いたい志向。外部補助で到達すると、結果が出ても手応えが薄れる。

### [valueLocus-5]

- **質問文**: 決定案には、何かしら自分の関与やアイデア、こだわりの爪痕を残したい。
- **評価ラベル**: [max]残したい/[min]決まれば純粋に満足
- **コンセプト**: オリジナリティを発揮したい内在的満足の傾向。

### [valueLocus-6]

- **質問文**: どこに出すか、どう伝えるか、どう根回ししておくかで結果を変えることに関心がある。
- **評価ラベル**: [max]とても関心がある/[min]それほど関心がない
- **コンセプト**: 流通・配置・合意形成の設計によって結果をコントロールする関心。

### [valueLocus-7]

- **質問文**: 自分の努力や成果にリアクションがない状態は、正直モチベーションを保ちにくい。
- **評価ラベル**: [max]認められるとやる気がでる/[min]自分のためにやるので関係ない
- **コンセプト**: 自分の行為や成果に対して外から反応が返ることで価値や駆動が補強されやすい特性。

### [valueLocus-8]

- **質問文**: 意見を出し合う場面ではまず様子をうかがいたい。人の考えを聞いてから調整できるほうが安心。
- **評価ラベル**: [max]みんなの意見と調和させたい/[min]自分は自分なので気にならない
- **コンセプト**: 自分単独で押し出すよりも、他者の考えや場の反応を見て接続点を探りながら調整できる状態に安心を感じる志向。

### [valueLocus-9]

- **質問文**: 決まっていないことが多く進捗がみえない状態、相談できる素材が少なく反応を見ながら調整できない状態はストレス。
- **評価ラベル**: [max]不安でストレス/[min]自由度が高くてむしろ楽しみ
- **コンセプト**: まだ外に出せる形や、すり合わせの素材がない状態を扱いにくく感じやすい。反応や接続の足場がない未形成状態へのストレス。

### [valueLocus-10]

- **質問文**: そんなにベストを尽くせてないものでも、相手が喜んだらそれはそれで良かったと思う。
- **評価ラベル**: [max]そう思う/[min]自分が納得できないと良かったと思えない
- **コンセプト**: 内容の熟し方や自己基準の完成度よりも、相手側で価値が成立したかどうかを重く見やすい志向。外部で喜ばれたこと自体が価値の成立条件になりやすい。

### [boundary-1]

- **質問文**: 「話題だから美味しい」「有名だからおしゃれ」という評価のしかたは違和感がある。
- **評価ラベル**: [max]違和感を感じる/[min]どこに違和感？
- **コンセプト**: 別のカテゴリが混線した評価への違和感があるかどうか。

### [boundary-2]

- **質問文**: どこの誰が発信したかよりも、伝えているものが何かが気になる。
- **評価ラベル**: [max]内容に注目する/[min]誰が言ったかは大事
- **コンセプト**: ネームバリューや立場と価値という別々のものが影響し合うことに違和感があるかどうか。

### [boundary-3]

- **質問文**: 応援しているものがマイナーだったり弱かったりアンチが多くても、自分なりに好きなポイントを持っていれば特段評価が落ちたりはしない。
- **評価ラベル**: [max]好きだし応援する/[min]評価が揺らぐ
- **コンセプト**: 人気や強さという軸と、自分の嗜好という軸、別軸の評価を切り分けて維持できるかどうか。

### [boundary-4]

- **質問文**: 自分がおすすめしたものへ相手は興味がなくてもかまわない。
- **評価ラベル**: [max]ぜんぜん平気/[min]ややさみしい
- **コンセプト**: 自分の評価と他者の関心を分離して扱える境界の安定性。

### [boundary-5]

- **質問文**: 当たり前ってそもそも何なんだろう。
- **評価ラベル**: [max]考えれば考えるほどわからない/[min]それとなくわかる
- **コンセプト**: 定義が曖昧な規範語を自明に受け取り難く、細分化して客観視したくなる志向。

### [boundary-6]

- **質問文**: 多少のことがあっても、人間関係でまとめるとよしなにできて安心。
- **評価ラベル**: [max]だいたいそう/[min]多少が何かによるんじゃない？
- **コンセプト**: 関係性で関係性ではないことの評価も動かす混線をよしとするかどうか。

### [boundary-7]

- **質問文**: 人気や権威のある人物との関係は、それが小さな接点でも特別な意味を感じる。
- **評価ラベル**: [max]特別感がある/[min]薄い関係でも意味あるかな？
- **コンセプト**: 人気や権威という規範的象徴への憧れとそこへの価値の収束を心地よく感じる志向かどうか。

### [boundary-8]

- **質問文**: 方向性が分かれそうな取り組みほど、代表がリードする体制がわかりやすくて安心。
- **評価ラベル**: [max]まとまりは大事/[min]分かれたらリードより議論では？
- **コンセプト**: 代表への委譲や一体感によって安心を得る融合志向。

### [boundary-9]

- **質問文**: 記念写真や記念日、推し色など、親密さを確認するものは気分が上がる。
- **評価ラベル**: [max]気分が上がる/[min]特に何も感じない
- **コンセプト**: 記号や共有物を通じて関係の近さや一体感を確認できることに価値を感じやすい。

### [boundary-10]

- **質問文**: 決をとるとき、賛成か妥協でまとまらず、反対したままの人が残るとモヤモヤしてしまう。
- **評価ラベル**: [max]モヤモヤする/[min]決まれば気にならない
- **コンセプト**: 結果よりも同質性を気にかける傾向があるかどうか。
